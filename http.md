# 1.HTTP 请求交互的基本过程

<pre>
                    >>>>>>>>>>>>>>>>>
        客户端（浏览器）              服务器
                    <<<<<<<<<<<<<<<<<
</pre>

- 1.1 浏览器向服务器发送的HTTP请求也叫请求报文。<br />请求报文有三部分组成：请求行、请求头、
请求体
- 1.2 服务器接受到请求后，处理请求，返回给客户端的响应又叫响应报文。<br />响应报文也有三部分组成：响应状态行、响应头、响应体
- 1.3 浏览器接收到响应体，解析响应体显示到页面。
